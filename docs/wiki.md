# ST_entanglement Analysis Wiki

This wiki is meant to serve as an analysis-facing knowledge base for the project.
Its purpose is to collect the practical context around the analysis in a form that is easier to
browse than a single long page.

## How To Use This Wiki

Use this page as the primary landing page and entry index.
Each topic below has its own dedicated page so that references, framework notes, and analysis
conventions can evolve independently.

## Notes On Scope

Some entries document what is already configured in `ST_entanglement` today.
Other entries document recommended CMS POG inputs together with the corresponding ColumnFlow
interfaces that exist upstream but are not yet fully enabled in this analysis module.

That distinction is intentional: the wiki should help separate

- current analysis behavior
- available framework support
- future integration work
